# RoNR
Risk the Night till it Rain

Risk of Nightreign is a mod for Elden Ring: Nightreign.

The goal of this mod is to increase build variety, run variability and ultimately replayability by adding roguelike elements inspired by Risk of Rain.

[Nexus Page](https://www.nexusmods.com/eldenringnightreign/mods/424)  

[Discrot](https://discord.gg/py5Y6cSUdf)

This mod requires [Mod Engine 3](https://github.com/garyttierney/me3/releases)

To access online features with this mod, you need [Nightreign Seamless Coop](https://www.nexusmods.com/eldenringnightreign/mods/3)
